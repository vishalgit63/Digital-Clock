A clean, customizable digital desktop clock application that displays the current day, date, and time (hours, minutes, seconds, AM/PM). The UI in the screenshot shows a stylized font and color scheme suitable for a lightweight desktop utility.

## Features
- Shows weekday and date (e.g., Wed, 01-04-2026)
- Large digital time display with hours, minutes, seconds and AM/PM
- Stylable fonts, colors, and layout
- Lightweight, runs as a simple desktop window or in a browser (see implementations)
- Easily customizable labels for Hr / Min / Sec
